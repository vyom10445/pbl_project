Automatic Handwritten Answer Sheet Digitizer Using Computer Vision & LSTM-OCR

This project aims to develop a system that converts handwritten exam answer sheets into structured, editable digital text using computer vision preprocessing and LSTM-based OCR, enabling faster evaluation and improved consistency.

Problem Statement

Universities still evaluate handwritten answer sheets manually, which is time-consuming and inconsistent. Existing OCR systems perform well on printed text but struggle with exam-style handwriting containing noise, skew, and irregular spacing. There is a need for a structured preprocessing and OCR-based system tailored specifically for academic answer sheets.

Literature Review

Tools like Tesseract OCR and cloud-based APIs provide handwriting recognition but lack optimization for large-scale academic evaluation and noisy exam sheets.

Research Gap

Most OCR pipelines are not designed specifically for exam sheet structure. This project integrates preprocessing (deskewing, segmentation) with LSTM-based OCR and confidence scoring for academic reliability.

System Methodology

Input & Preprocessing

Scanned or camera-captured sheets are converted to grayscale, denoised, thresholded, and deskewed using OpenCV.

OCR & Extraction

LSTM-based Tesseract OCR extracts handwritten text. Segmentation improves line-level recognition accuracy.
