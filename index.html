Automatic Handwritten Answer Sheet Digitizer Using Computer Vision & LSTM-OCR

This project develops a system that converts handwritten exam answer sheets into structured digital text using computer vision preprocessing and LSTM-based OCR, enabling faster evaluation and improved consistency.

Problem Statement

Manual evaluation of handwritten answer sheets is time-consuming and inconsistent. Traditional OCR systems perform well on printed text but struggle significantly with exam-style handwriting that contains noise, skew, uneven lighting, and irregular spacing. There is a need for a structured preprocessing and OCR-based system tailored specifically for academic answer sheets.

Literature Review / Market Research

Existing OCR engines such as Tesseract and cloud-based APIs provide handwriting recognition but are primarily optimized for printed text. Research in LSTM-based OCR models shows improved performance for cursive writing, but academic sheet-specific preprocessing pipelines are rarely implemented.

Research Gap / Innovation

Most OCR systems apply static preprocessing. This project introduces adaptive preprocessing selection combined with confidence-based evaluation, specifically targeting exam-sheet structures to improve reliability in academic contexts.

System Methodology

Input & Preprocessing

Grayscale conversion, noise removal, thresholding, and deskewing using OpenCV.

OCR Engine

LSTM-based Tesseract OCR extracts handwritten text with word-level confidence scoring.
